# Using ELM techniques to predict data center VM requests

## 简介

数据中心的预测模型可以进行CPU、内存、VM请求率等一系列指标的预测。本文使用预测式模型，与k-means和Extreme Learning Machines结合。目标是基于历史数据预测流量信息，使用数据集为Google trace data的29天数据。

## 评价

很单纯的一篇预测式的文章，没什么好说的。
